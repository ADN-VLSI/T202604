
1. Main Goal
----------------------

- Check if I can write and read data correctly from the registers.
- Make sure the Error flag turns ON when doing something wrong.



2. What to Test (Checklist)
-------------------------------------

- Reset: Check if everything clears to `0` when reset is pressed.

- Read-Only (RO): `reg0` to `reg3` should only allow reading, no writing.

- Read-Write (RW): `reg4` to `reg7` should allow both writing and reading.

- Error: Trying to write into the first 4 registers (RO) must force `error_o = 1`.



 3. Step-by-Step Test Steps
 -----------------------------------

Step 1: Press Reset (`arst_ni = 0`)  check if outputs become zero.
Step 2: Put some random data into `reg0` and `reg1`  read them back to see if the data shows up.
Step 3: Write new data into `reg4` and `reg7`  read them back to see if the data was actually saved.
Step 4: Try to write into `reg2` (which is Read-Only)  check if `error_o` immediately goes high.
